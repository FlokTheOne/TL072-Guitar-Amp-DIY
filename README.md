This Guitar amp circuit was demonstrated by KiCad and the op-amps used in this circuits were TL072 (as the preamp) and TDA2030 (as the power amp).

The project's name is "Attempt" if you didnt notice.

## Components
- 1x TL072
- 1x TDA2030A module
- 1x 12V adapter
- Passive components (resistors, capacitors, you name it)
- 1x 100k Potentiometer knob for gain control
- 2x 6.35mm (0.25 inch) audio jacks (Guitar input and output)
- 1x Speaker (preferably 10W)
- 1x DC Barrel Jack
